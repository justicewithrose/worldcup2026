# World Cup 2026 Calendar

A simple website built by football fans to help other fans follow the FIFA World Cup 2026.

## Features

- View all World Cup fixtures
- Filter by team or group
- Add matches directly to your calendar
- Receive reminders before kickoff
- Mobile friendly

## Live Website

https://worldcup2026.justicewithrose.com

## Why We Built This

As football fans from India, we realized that World Cup matches in the USA, Canada and Mexico would happen at unusual hours for many fans around the world.

We wanted an easy way to add matches to our calendars and never miss a game.
